心之所向，持之以恒

发表于 2017-06-20 | 分类于 随笔 | | 阅读次数

我是今年过完年(2017年初)才有写技术、生活博客的习惯，写博客的初衷就是想把自己平时工作中的开发技能分享出来，给遇到相同问题的童鞋提供一点线索。毕竟工作了几年有一定的开发经验，而且身边的很多同事也有自己的博客，所以自己也就加入到博客大军中。后来写了几篇博客之后对博客有了更深入的了解：当你想写一篇博客时（一般是自己有一定见解的内容），往往会通过官方文档或者比较优秀的文章来补充自己在这个知识模块中认识的不足，以便完成这个模块内的体系结构，而这个过程其实就是一个学习的过程。当你对该模块知识整体把握后就可以通过”演示例子“+”说明“+”整体总结“将博客写下来，按照学习金字塔的概念这个过程是大脑的主动学习将会加深知识的记忆。在写博客之前我学习的方式也很简单，通常是通过代码写个demo把新知识功能演示下来，然后就没有然后了，并没有对该知识进行总结，这其实并不利于以后对知识的复习也没有训练自己的表达能力。所以写博客除了能分享技术给别人看之外，其本身就是集学习、总结、复习于一身的自我提升！

编写该篇博客的目的主要是明确下写博客的重要性，阐述自己对写博客的观点、自己现在的状况。对自己以此勉励坚持写博客的习惯，对读者给予提醒有这么个好的方式来记录成长。

加入博客到现在也有小半年的时间了，写了大概十几篇博客，其实产量不算多。毕竟是在工作之余写写，希望以后每周或两周能够写一篇文章，来勉励自己不断学习。这半年写的博客主要是对开发经验技巧的总结，像数据持久化总结，iOS多线程的使用总结，这些总结性的博客往往会花费很长时间。因为这类的文章内容比较多，演示demo也很多需要花费很长时间去归类、小结、总结。其中我记得多线程的总结我大概花费了3周的时间才给写完。当你写下很大篇幅的博客时内心满满的成就感让人身心愉悦😆。在没有写博客之前周末可能更多的是看视频，玩游戏度过了，而当自己有写博客的打算后周末通常是休息娱乐和学习相结合，通常劳逸结合往往能带来事半功倍的效果何乐不为呢！

“大道至简，贵在坚持“，做好一件事情绝非一时之功。就像我入门iOS开发一样，从大二的暑假到大三的上半学期一直都在坚持学习，那是一个非常枯燥漫长的时间，自己一个人从基础的Objective-C语言开始到各个UI控件的使用到后来的网络编程慢慢的学习完成。现在回想起来，也是佩服自己能够坚持那么长时间，尤其是炎热的暑假自己一个人在寝室敲着代码，满满的回忆☺️。入门了不代表不需要学习新的知识了，在互联网这种行业，技术日新月异，不坚持学习是要被时代给抛弃的。既然我选择以博客的形式来记录我的学习脚步，那我就要坚持下去，以提升自己的认知。就如标题“心之所向，持之以恒”，一步一个脚印往前走！
